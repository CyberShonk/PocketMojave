Compatibility is not a promise

Listed results depend on game version, mod list, driver version, GameNative version, DML version, storage setup, and user configuration. Do not treat one result as a guarantee for every device or every future version.

What counts as proven working?

  • The game launches from GameNative.
  • The main menu loads.
  • The guide target reaches Goodsprings or another stable New Vegas start area.
  • The user can play for at least 15 minutes.
  • Crashes, graphical bugs, audio bugs, input bugs, thermal behavior, and FPS are reported honestly.
  • The report includes enough version details for another user to reproduce the setup.

Memory warning

Use 2 GB memory and 2 GB video memory unless your Android device has 16 GB of RAM. If your device has 16 GB of RAM, you can leave the memory setting at 4 GB. Lower-memory devices may crash, stutter, or fail to launch if the profile reserves too much memory.

Status labels

Verified working

Launch, menu, in-game load, and 15-minute play test passed with useful version details.

Working with issues

Playable, but crashes, graphics, audio, input, heat, or performance problems are documented.

Boots only

The game opens or reaches the menu, but gameplay is not proven.

Unstable

Runs sometimes, but crashes or breaks often enough that it should not be called working.

Not working

Fails to launch, fails to reach the menu, or cannot load into the required start area.

Needs retest

The report is old, incomplete, or affected by changed GameNative, Turnip, DML, DXVK, Proton/Wine, or FEXCore versions.

What not to submit as working

  • A report that only says "it works" with no version numbers.
  • A setup that only opens the launcher but never loads in-game.
  • A result from a different GameNative build, DML version, or driver version without saying so.
  • A result where the game only runs once and then crashes on relaunch.
