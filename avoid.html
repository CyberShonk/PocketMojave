Tools and Mods to Avoid

No author drama
This page is not for dunking on mod authors. It is about keeping the Android guide clean, easy to test, and beginner-friendly.

Tools to avoid for this guide

Tool / practiceStatusWhy
Old mod managers as the main setupAvoidDo not build this guide around FOMM, old NMM, or random manual Data folder piles. Use DML for the Android side.
New Vegas Stutter RemoverAvoidReplaced by modern tick/performance fixes. It is not a good fit for this guide.
NVAC as a normal recommendationAvoidIt can hide crashes that should be reported and fixed. Use Crash Logger for useful reports.
ENB / ENBoostAvoidNot the target renderer path for Android/GameNative.
Random auto-purge/cache cleaner modsAvoidThey can add stutter and make troubleshooting harder.
One-click desktop modlistsAvoidThey assume desktop MO2 paths, desktop LOD, and desktop GPU behavior.

Major balance overhaulsPersonal choiceNot part of this guide because difficulty and DLC changes are taste-based.

Safe modding rules

  • Read the mod page and requirements.
  • Install one section at a time.
  • When something breaks, undo the last change first.
  • Do not add ten mods after a failed test.
  • If a mod claims performance improvement, benchmark it on your device.
  • Keep root tools, DML mods, and generated outputs clearly separated.
