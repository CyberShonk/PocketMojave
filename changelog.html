Changelog

Mod-list changes only. This page is for added mods, removed mods, load-order-impacting changes, and save-safety notes.

Small website fixes, typo fixes, layout changes, and setup-command changes are intentionally not tracked here.

How to read this page

Save safe means the update should be okay for an existing save if you follow the notes. New game recommended means the update changes enough of the list that a fresh save is the cleanest test. New game required means do not add the update mid-playthrough unless you know exactly what you are doing.
