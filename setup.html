DML setup

Create a fresh profile in Droid Mod Loader for New Vegas.

  1. Open Droid Mod Loader.
  2. Create or select a Fallout: New Vegas profile.
  3. Name it Pocket Mojave.
  4. Set the deployment target to the game's Data folder.
  5. Confirm that the target contains FalloutNV.esm.
  6. Do not select the game root as the Data target.
  7. Install mods in the order shown in the mod list.
Order typeWhat it controls
Mod priorityLoose file conflicts. Higher priority wins loose file overwrites.
Plugin orderESM and ESP load order. This controls record conflicts and masters.
Do not mix these up
A mod can be in one place in the DML mod list and have a plugin in a different place in the plugin list. They solve different problems.

BethINI Pie

Use BethINI Pie to set clean INI values before the mod list gets bigger.

  1. Run the New Vegas launcher once in the GameNative prefix so the INI files exist.
  2. Open BethINI Pie on PC or inside a Windows environment that can see the same New Vegas INI files.
  3. Select Fallout: New Vegas.
  4. Use BethPresets.
  5. Choose Medium.
  6. Set texture quality to max quality. Do not leave texture mip skip on a low preset.
  7. Keep resolution at 1280x720 for the guide target.
  8. Save the INIs.
Recommended audio setting
Enable audio multithreading. It can help New Vegas avoid small audio stalls on CPU-limited setups.
Audio cache caution
The audio cache settings can reduce loading stutter for some setups, but they can also be unstable in Windows containers. Change them only after the guide works, and keep notes so you can revert.

Suggested FalloutCustom.ini audio block

[Audio]
bMultiThreadAudio=1
bUseAudioDebugInformation=0
; Optional test only. If audio gets unstable, remove these two lines.
iAudioCacheSize=16384
iMaxSizeForCachedSound=2048

After setup check

  1. Launch with no mods after BethINI Pie saves the INIs.
  2. Start or load the Doc Mitchell test save.
  3. Walk outside, save, quit, and reload.
  4. Do not install mods until this still works.
