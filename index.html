Shonk's Pocket Mojave

A mobile-friendly Fallout: New Vegas guide for Android handhelds, GameNative, and Droid Mod Loader.

This is a living guide. The goal is simple: install the best core New Vegas mods, keep the setup easy to understand, and make Android modding less like copying files in the dark.

Start setupOpen mod listLoad order help

Built for handhelds

Targets Android phones and handhelds first. Desktop guides are useful, but mobile has different storage, driver, and testing problems.

DML first

Mod installation is written around Droid Mod Loader, with PC tools used only where they still make sense.

No LOD pass yet

LOD generation and large visual toolchains are separate projects. This guide keeps them out until they have a clean mobile path.

Before you start

Read the guide sections in order. Do not skip the vanilla test because the game opened once.

Vanilla check

  1. Launch Fallout: New Vegas with no mods.
  2. Start a new game and reach Doc Mitchell's house.
  3. Make a manual save.
  4. Quit the game completely.
  5. Relaunch and load that save.
