Shonk's Pocket Mojave

A mobile-friendly Fallout: New Vegas guide for Android handhelds, GameNative, and Droid Mod Loader.

This is a living guide. The goal is simple: install the best core New Vegas mods, keep the setup easy to understand, and make Android modding less like copying files in the dark.

Built for handhelds

Targets Android phones and handhelds first. Desktop guides are useful, but mobile has different storage, driver, and testing problems.

DML first

Before you start

Before you start: use the correct GameNative build

Use the current GameNative 1.0.0 standard APK/release path shown in this guide unless a later guide update says otherwise.

A specific older v1.0.0 prerelease modern Android 11+ APK caused problems for this guide because its notes said it had no custom game support and no D: drive access. These guides rely on custom games and shared-storage style setup, so the wrong APK can make the guide fail before mods are involved.

If custom games do not work, or the Windows container cannot see the expected drive, check which GameNative APK/build you installed before changing random settings.

Install New Vegas in shared storage

Do not put Fallout: New Vegas inside locked GameNative App Storage for this guide. DML needs practical access to the same game folder GameNative launches.

/storage/emulated/0/PcGames/Fallout New Vegas/

DML should deploy to the game Data folder, not the game root.

Vanilla check

  1. Launch Fallout: New Vegas in GameNative with no mods.
  2. Reach Goodsprings/Doc Mitchell.
  3. Make a manual save.
  4. Quit the game completely.
  5. Relaunch and load that save.
