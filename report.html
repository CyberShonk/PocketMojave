Fast troubleshooting

  1. Confirm the game still launches through nvse_loader.exe.
  2. Check for missing masters in the plugin list.
  3. Check that the exported plugins.txt and loadorder.txt are in the GameNative prefix path the game actually reads.
  4. Disable the last mod installed and test again.
  5. If the issue remains, disable the last whole section and test again.
  6. If a UI menu breaks, stop adding mods and fix the UI section first.
  7. If audio breaks after cache edits, remove the cache edits and keep audio multithreading only.

What not to do

  • Do not reinstall the whole game before checking the last mod change.
  • Do not keep adding mods after a failed launch.
  • Do not change Proton, Turnip, DXVK, BethINI, and mods all in one test.
  • Do not assume desktop fixes apply cleanly to Android.
