Mod List

Install the sections in order. The layout matches the companion guide, but the actual mod choices stay specific to Pocket Mojave.

Before downloading mods

Install sections in order. Use the listed file unless the note says otherwise. When a mod has optional files or special download choices, the note is listed directly under that mod.

Root tools warning

Some tools belong in the game root next to FalloutNV.exe, not inside Data. This applies to both guides because xNVSE, the 4GB patcher, and some helper tools are root-level tools.

Required PC-side preparation

Run these on the Windows PC copy of Fallout: New Vegas before relying on the Android/GameNative copy.

Ultimate Edition ESM Fixes Remastered

Recommended
Main fileMain Files - Ultimate Edition ESM Fixes Remastered
Run onWindows PC copy first

This is a PC-side preparation tool, not an Android app. Run it on the Windows PC copy of Fallout: New Vegas before relying on the Android/GameNative copy. After using it, launch the PC copy once and make sure the game still starts before continuing.

FNV BSA Decompressor

Recommended
Main fileMain Files - FNV BSA Decompressor

This is a PC-side preparation tool. Use it before transferring or relying on the Android copy. If the prepared PC copy is already broken, stop and fix that before troubleshooting GameNative or DML.

Notes for optional files or special downloads

When a mod has optional files or special download choices, the note is listed directly under that mod. Do not install every optional file just because it exists.

What not to change here

Do not add TTW-only requirements to Pocket Mojave. Do not change Proton, DXVK, Turnip, DML, and the mod list all at the same time.

Next step

After this page is installed and tested, continue to the Load Order page. Use DML to write the activation files, then separately verify the timestamp-based New Vegas plugin order in the actual Data folder.

Checkpoint

  1. Deploy in DML.
  2. Write and copy the current plugin files into GameNative AppData.
  3. Verify the ESM and ESP timestamps in the shared-storage Data folder.
  4. Launch through GameNative.
  5. Reach the main menu and load into Goodsprings.
